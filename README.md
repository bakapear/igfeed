# igfeed
### On new Instagram post -> Send Discord Webhook

Pretty simple and sassy~

## Requirements
- NodeJS

## Running
1. Make sure you installed all packages with `npm install`
2. Configure settings in `config.json`
3. Start with `npm run start`
